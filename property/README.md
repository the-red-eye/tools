# Property Monitoring Tools

Tools for monitoring real estate listings in Portugal.

## Supported Platforms
- Idealista
- Imovirtual
- Casa Sapo
- RE/MAX
- Century 21
- ERA
- IAD

## Usage

These tools use browser automation (Playwright) to scrape listings.

```bash
pip install playwright
playwright install chromium
```

See individual scripts for specific usage.

## Note

Be respectful of rate limits. These tools are for personal use monitoring.
